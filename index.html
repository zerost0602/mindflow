본인의 Supabase 프로젝트 정보를 입력하여 데스크톱, 모바일 어디서나 접속해 실시간으로 데이터를 관리하세요. 데이터베이스 구축 방법은 아래 가이드를 참고하세요.

Supabase 연동 가이드

  1. Supabase.com에 접속해 무료 가입합니다.
  2. 새 프로젝트(New Project)를 임의의 이름으로 생성합니다.
  3. 생성 직후 화면에 제공되거나, Project Settings -> API에서 Project URL 및 API Key (anon/public)를 확인해 복사합니다.
  4. 좌측 메뉴에서 SQL Editor를 열고, New query를 누릅니다.
  5. 우측 [implementation_plan.md]에 안내된 Supabase SQL 스크립트(테이블 생성 쿼리)를 복사해 SQL Editor에 붙여넣은 뒤 Run 버튼을 눌러 실행합니다.
  6. 이 화면으로 돌아와 URL과 Key를 붙여넣고 [연동 설정]을 완료하면, 기존의 오프라인 데이터가 내 개인 Supabase 데이터베이스로 즉시 마이그레이션 및 동기화됩니다!
